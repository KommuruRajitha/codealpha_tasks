Here's a simple Java chatbot program that interacts with the user via the console. The chatbot will recognize a few specific inputs and give predefined responses. If it doesn't recognize the input, it will respond with a generic fallback message.

java
import java.util.Scanner;

public class SimpleChatBot {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String userInput;
        System.out.println("ChatBot: Hello! I'm your simple chatbot. Type 'bye' to end the chat.");

        while (true) {
            System.out.print("You: ");
            userInput = scanner.nextLine().toLowerCase();

            if (userInput.contains("hello") || userInput.contains("hi")) {
                System.out.println("ChatBot: Hi there! How can I help you today?");
            } else if (userInput.contains("how are you")) {
                System.out.println("ChatBot: I'm just a bot, but I'm doing fine! Thanks for asking.");
            } else if (userInput.contains("what is your name")) {
                System.out.println("ChatBot: I'm ChatBot, your friendly assistant.");
            } else if (userInput.contains("bye")) {
                System.out.println("ChatBot: Goodbye! Have a great day!");
                break;
            } else {
                System.out.println("ChatBot: Sorry, I didn't understand that.");
            }
        }

        scanner.close();
    }
}

  • Responds based on the content of the input using contains() and toLowerCase().

  • Exits when the user types "bye".
